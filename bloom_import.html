Import and manage patient aligner cases from bloomaligner.fr. Supports both CSV (case status updates) and Excel (shipping + aligner details) file formats.
📂 CSV Import
Export from bloomaligner.fr and import here. Updates case statuses, overdue flags, doctor/distributor info for all cases.
📊 Excel Import
Import the Bloom Excel file. Reads SHIPPING sheet (box numbers, dates) and ALIGNER sheet (upper/lower counts, order type).
Cases Tab
Full list of all Bloom cases with sortable columns and filters: status, doctor, date range, search.
Aligners Tab
All aligner orders with patient name, upper/lower counts, package type, shipping info.
Analytics Tab
Charts showing cases by status and aligners by order type. Uses same filters as the tables.
History Tab
Log of all status changes per case over time.
Imports Tab
Log of all CSV/Excel import events with timestamps.
🔵
Bloom Import
Role: Production Manager / Order Manager
Sync patient aligner cases from bloomaligner.fr into the system. Two import formats: CSV for case status updates, Excel for aligner and shipping details.
Process Flow
1
CSV Import
Export from bloomaligner.fr → import CSV here → updates all case statuses, overdue flags, doctor/distributor info.
2
Excel Import
Import Bloom Excel file → reads SHIPPING sheet (boxes, dates) + ALIGNER sheet (upper/lower counts, order type).
3
Cases tab
Full sortable/filterable list of all cases. Filter by status, doctor, date range. Search by name or case ID.
4
Aligners tab
All aligner orders with patient, upper/lower counts, package type, shipping info.
5
Analytics tab
Charts: cases by status, aligners by order type. Uses same filters as tables.
